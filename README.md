# Most Common Issues

> If you can't solve the issue ping Head Support for assistance

## VBS/Hyper-V doesn't turn off

Provide link to [guide](https://github.com/KilluaUwU/Disable-Hyper-V) and remind to delete passwords
> Turning off not required for games other than genshin

## BSOD GENSHIN

1. Ask for screenshots of task manager and System tab from launcher
2. Virtualization disabled - ask to turn it on from BIOS (Intel Virtualization Technology(VMX)/VT-x/VT-d for intel cpu, SVM mode/AMD-V for amd cpu)
3. Ask to fix issues if launcher is not all green
4. Antiviruses should be deleted, Anticheats - closed

## BSOD other uni games

1. Ask launcher screenshots(Virtualization is not required for those)
2. VBS/Hyper-V can be red, not required for other games
3. If there are antivirus they should delete it
4. Anticheats should be closed
5. If issue persists - ask to turn off Exploit Protection tab completely and restart pc
6. If issue still persists - ask to "win + r - msconfig - services - tick hide Microsoft services - disable all - restart pc and try"

## Game Crash/No menu (any Uni game)

1. Ask them to set game to borderless screen mode
2. If didn't help, ask what GPU does user have
3. If it's NVIDIA 40/50 series, ask if they have smooth motion enabled in nvidia app
4. Ask to disable if it's enabled

## Failed to place required callback

Usually caused by Exploit Protection/3rd party antivirus/Anticheat

Ask user to turn these off/delete antivirus

If it didn't help - "win + r - msconfig - services - tick hide Microsoft services - disable all - restart pc and try"
